Commands
========

.. program:: tailwind

.. option:: install

    Installs the tailwind compiler.

.. option:: start

    Starts the tailwind compiler.

.. option:: build

    Builds tailwind for production.

.. program:: setfingerprint

.. option:: -id <int>, --employee-id <int>

    An employee id.

.. option:: -c <str>, --code <str>

    A fingerprint code.

    Builds tailwind for production.
